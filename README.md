# 🎨 Abhishek Maurya - Portfolio

A modern, interactive portfolio website featuring a 3D avatar, smooth animations, and working contact form.

## 🛠️ Tech Stack

React 18 • Vite • Three.js • Framer Motion • Tailwind CSS • Web3Forms

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/abhishek-maurya576/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Create .env file and add your Web3Forms API key
# VITE_WEB3FORMS_ACCESS_KEY=your_key_here

# Start development server
npm run dev

# Build for production
npm run build
```

## 🔑 Setup Contact Form

1. Get free API key from [web3forms.com](https://web3forms.com/)
2. Create `.env` file in root directory
3. Add: `VITE_WEB3FORMS_ACCESS_KEY=your_key_here`

## 📦 Deployment

**Netlify/Vercel:** Add `VITE_WEB3FORMS_ACCESS_KEY` environment variable in dashboard
