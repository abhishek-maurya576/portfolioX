# 🎨 Abhishek Maurya - Portfolio

Modern, interactive portfolio with 3D avatar, animated skill folders, and smooth animations.

## 🛠️ Tech Stack

React 18 • Vite • Three.js • Framer Motion • Tailwind CSS • Lucide Icons

## ✨ Features

- **3D Avatar** - Interactive Three.js character
- **Animated Skill Folders** - Hover-reveal skill cards with icons from skillicons.dev
- **Neumorphic UI** - Modern soft-shadow design
- **Contact Form** - Web3Forms integration
- **Responsive** - Desktop & mobile optimized

## 🔑 Contact Form Setup

1. Get API key from [web3forms.com](https://web3forms.com/)
2. Create `.env`: `VITE_WEB3FORMS_ACCESS_KEY=your_key`
